# SilverShore Website Pages

This repository is a static GitHub Pages mirror for SilverShore website artifacts.

## Ownership

- Owner surface: SilverShore website fallback
- GitHub: `https://github.com/shanegardner405-arch/silvershore-website-pages`
- Visibility: public
- Deployment surface: GitHub Pages static hosting
- Canonical dynamic website repo: `https://github.com/shanegardner405-arch/silvershore-website`

## Status

This repo is fallback/public-display infrastructure only. It exists for static public-safe mirrors, static documentation, or temporary display artifacts when Vercel quota or staging access blocks normal preview work.

It is not the canonical dynamic website repo and must not be treated as a replacement for Vercel app behavior.

## Public-Safety Boundary

Allowed:

- static public marketing pages
- public-safe images and PDFs
- static status or preview artifacts

Not allowed:

- credentials or environment values
- private client data
- internal AIOS strategy
- operational receipts not intended for public viewing
- dynamic app behavior, API routes, auth flows, cron behavior, or database-backed features

## Verification

Use GitHub Pages readback and static file inspection. Do not claim this proves Vercel production or preview behavior.

## What Not To Use This Repo For

- production dynamic app proof
- private SilverShore operating docs
- client workspaces
- source-of-truth website development
